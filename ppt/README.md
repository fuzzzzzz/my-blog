## 第二页：学校简介（01）

### 历史沿革

| 年份 | 里程碑 |
|------|--------|
| **2002年** | 九江大学、九江师专、九江医专、九江教育学院四校合并，正式成立本科院校 |
| **2008年** | 通过教育部本科教学工作水平评估 |
| **2013年** | 获批江西省首批"2011协同创新中心" |
| **至今** | 多学科协调发展的应用型本科院校 |

### 地理位置
- 📍 江西省九江市濂溪区前路二路
- 🏔️ 庐山北麓 · 长江之滨 · 鄱阳湖畔
- ✈️ 距庐山机场约 20 分钟车程
- 🚄 京九铁路、昌九城际在此交汇

### 学校定位
> 以"立足九江、服务江西、面向全国"为办学宗旨，坚持"以学生为中心、以质量为生命、以特色求发展"的办学理念，致力于建设成为特色鲜明的高水平应用型大学。

## 第六页：国际交流与合作（05）

### 发展路径
```
中外合作办学项目 → 海外实习与游学 → 国际认证与交流
```

### 四大方向

#### 🌍 海外合作院校
与美国、英国、澳大利亚、新西兰、日本、韩国等多个国家的高校建立合作关系，开展学分互认、学生交换等项目。

#### 🎓 留学生教育
积极招收来华留学生，开设多语种授课专业，已培养来自 **20余个** 国家的留学生。

#### 📖 学术交流
定期举办国际学术会议，邀请海外知名学者来校讲学，选派教师赴境外研修。

#### 💻 双语教学
推进双语课程和全英文授课专业建设，引进国际优质课程资源。

## 第七页：未来发展规划（06）

### 战略目标
> 到2035年，将学校建设成为**特色鲜明的高水平应用型大学**，成为区域人才培养、科技创新和文化传承的重要基地。

### 发展重点
1. **优化学科专业结构** — 强化工科和医科建设
2. **深化教育教学改革** — 创新人才培养模式
3. **加强科研平台建设** — 提升服务地方能力
4. **服务地方战略** — 对接九江市"工业强市"战略和江西省"2+6+N"产业布局

### 关键目标
- 学科建设：争创省级一流学科 **3-5个**
- 人才引育：新增国家级人才 **10+名**
- 科研突破：国家自然科学基金新突破
- 国际化：中外合作办学项目 **5+个**
