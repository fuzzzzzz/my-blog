# My Blog

基于 pnpm workspace 的个人博客项目，用于分享技术文章和生活随想。

## 项目结构

- `apps/` — 博客应用主目录
- `ppt/` — 展示/演讲相关内容
- `docker-compose.yml` — Docker 部署配置

## 本地开发

```bash
pnpm install
pnpm dev
```

## 部署

```bash
docker-compose up -d
```
